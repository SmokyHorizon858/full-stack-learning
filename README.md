# Full Stack Learning
This repository records my full-stack developement learning journey.
Although I learned full-stack before, this is a deeper learning journey.

## Phase 1: HTML & CSS Fundamentals
What to learn
HTML:
- Headings, paragraphs, links, images
- Forms and inputs
- Tables
- Semantic tags: header, main, section, article, footer

CSS:
- selectors
- Box model
- Flexbox
- Grid
- Responsive design
- Basic animations

Success Standard:
Build a clean webpage without a framework.

## Phase 2: JavaScript Fundamentals
What to learn
- Variable: let, const
- Functions
- Arrays and objects
- Loops
- DOM manipulation
- Events
- localStorage
- fetch
- async / await
- Basic error handing

Success Standard
How the browser updates the page after user actions.
